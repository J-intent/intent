# Intent 编程语言

> 意图清晰，契约确定 | 心学为体，禅道为翼，马哲为用

Intent 是一门以**意图表达**为核心的编程语言，融合了东方智慧与西方工程思维。

## 🎯 核心理念

- **心学为体**：通过契约（前置/后置条件）明确意图
- **禅道为翼**：保持代码简洁，不被技术细节所困
- **马哲为用**：实践验证，迭代完善

## 🧠 哲学框架

每个Intent程序都是一次"知行合一"的实践：

1. **契约是良知**：明确能做什么，承诺什么结果
2. **实现是行动**：在代码中践行契约
3. **测试是检验**：通过实践验证真理
